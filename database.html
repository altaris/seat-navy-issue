Database modules¶

Mongodb¶

Database layer

Reference:

MongoEngine User Documentation

sni.db.init()[source]¶

Connects to the MongoDB instance.

Does not return anything. Any call to mongoengine will act on that connection. It’s magic.

sni.db.migrate() → None[source]¶

Runs various migration jobs on the database.

Should be called immediately after initializing the connection.

sni.db.migrate_ensure_root() → None[source]¶

Create root user if it does not exist.

sni.db.migrate_ensure_root_dyn_token() → None[source]¶

Create a dynamic app token owned by root, if none exist.

sni.db.migrate_ensure_root_per_token() → None[source]¶

Create a permanent app token owned by root, if none exist.

sni.db.migrate_ensure_superuser_group() → None[source]¶

Create the superusers group and makes sure that root is the owner.

sni.db.on_pre_save(_sender: Any, document: mongoengine.document.Document)[source]¶

If the document has a updated_on, sets it to the current datetime.
