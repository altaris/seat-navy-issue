Configuration facility¶

Reference¶

The configuration file is in YAML format, and named sni.yml by default.

  • database

    • authentication_source (default: admin)

    • database (default: sni)

    • host

    • password

    • port (default: 27017)

    • username (default: sni)

  • esi: Create an application at the EVE Developer Portal

    • client_id

    • client_secret

  • general

    • debug (default: false): Sets the app in debug mode.

    • host (default: 127.0.0.1): API server host.

    • port (default: 5000): API server port.

    • root_url: HTTP URL at which the API root is located (e.g. https://example.com/sni)

  • jwt

    • algorithm (default: HS256): A pyjwt supported algorithm, see here.

    • secret: Application secret. Generate with openssl rand -hex 32.

  • logging: Logging facility configuration, see the logging.config documentation.

  • redis

    • database (default: sni)

    • host

    • port (default: 6379)

Example¶

database:
    authentication_source: admin
    database: sni
    host: localhost
    password: password
    port: 27017
    username: sni
esi:
    client_id: 299bd5a9f432a7b9a584e91801cb95a3
    client_secret: 702e1947f7b876e73b790cbc1f13c9ad8a3b94e6
general:
    debug: true
    host: 0.0.0.0
    port: 80
jwt:
    algorithm: HS256
    secret: 3f961d92a82060c4375de5683e1a7f4d94bd48e3ba861c49a40708e3c9976ea1
logging:
    version: 1
    disable_existing_loggers: true
    formatters:
        standard:
            format: '%(asctime)s [%(levelname)s] %(name)s: %(message)s'
    handlers:
        default:
            level: INFO
            formatter: standard
            class: logging.StreamHandler
            stream: 'ext://sys.stdout'
    loggers:
        "":
            handlers:
                - default
            level: DEBUG
            propagate: false
redis:
    database: sni
    host: localhost
    port: 6379
