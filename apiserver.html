async sni.apiserver.post_auth_dyn(data: sni.apimodels.PostAuthDynIn, app_token: sni.dbmodels.Token = Depends(validate_header))[source]¶

Authenticates an application dynamic token and returns a state code and an URL at which the user can authenticate to the EVE SSO. Once that is done, SNI issues a GET request to the app predefined callback, with that state code and the user token.

async sni.apiserver.post_auth_per(app_token: sni.dbmodels.Token = Depends(validate_header))[source]¶

Authenticates an application permanent token and returns a user token tied to the owner of that app token.
