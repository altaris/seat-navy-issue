async sni.api.routers.callback.get_callback_esi(code: str, state: str)[source]¶

ESI callback. You should not manually call this.

Upon receiving a notification from EVE SSO, SNI redirects the client to the appropriate frontend callback, with the state code and user token in URL parameters.

Authenticates an application dynamic token and returns a state code and an URL at which the user can authenticate to the EVE SSO.
